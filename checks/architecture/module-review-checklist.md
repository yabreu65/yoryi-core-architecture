# Module Review Checklist

Checklist rapido para revisiones de modulos backend/frontend/AI.

- [ ] Limites de modulo claros.
- [ ] Contratos de entrada/salida definidos.
- [ ] Dependencias apuntan hacia adentro (no acoplamiento circular).
- [ ] Manejo de errores explicito.
- [ ] Seguridad y tenant context considerados.
- [ ] Observabilidad minima incluida.
- [ ] Tests de contrato definidos.
