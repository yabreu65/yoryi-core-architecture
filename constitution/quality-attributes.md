# Quality Attributes

## Prioridades

1. Seguridad de datos multi-tenant.
2. Confiabilidad operativa (SLO, error budget).
3. Trazabilidad y auditabilidad.
4. Escalabilidad de costos.
5. Velocidad de entrega sustentable.

## Minimos por servicio

- SLO definido.
- Logging estructurado.
- Trazas distribuidas.
- Alertas accionables.
- Owner explicito.
